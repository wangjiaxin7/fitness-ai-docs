# AI 健身教练 — 竞品分析

## 1. 分析目的

明确市场中已有产品的优劣势，找到差异化切入点，避免重复造轮子。

## 2. 竞品选择

选择 4 个有代表性的竞品，覆盖不同产品形态：

| 竞品 | 产品形态 | 用户规模 | 核心定位 |
|------|---------|---------|---------|
| **Keep** | App（运动+内容+电商） | 3 亿+ 注册用户 | 运动生活方式平台 |
| **薄荷健康** | App（饮食+体重管理） | 1 亿+ 注册用户 | 饮食管理工具 |

| **Hevy** | App（训练记录） | 400 万+ 用户 | 力量训练记录工具 |

## 4. 详细竞品分析

### 4.1 Keep

**产品概述**：国内最大的运动生活方式平台，覆盖运动课程、训练工具、运动电商、社交。

**优势**：
- 用户基数大，内容生态丰富（海量课程、直播）
- 品牌认知度高，"自律给我自由"深入人心
- 商业模式成熟（会员 + 电商 + 广告）
- 社交功能强（运动社区、排行榜）

**劣势**：
- AI 能力弱：训练推荐是模板化的，没有真正的 AI 对话
- 无动作评估：只有课程跟练，没有视频分析告诉用户做得对不对
- 功能臃肿：运动、电商、社交、内容混杂，核心训练体验被稀释
- 个性化不足：推荐基于简单标签，不基于训练历史深度分析

**对本产品的启示**：
- Keep 的弱点在于"有内容但没有指导"——用户跟着视频练，但不知道自己做得对不对
- 本产品可以切入这个空白：**AI 动作评估 + 个性化指导**

### 4.2 薄荷健康

**产品概述**：以饮食管理为核心的健康管理工具，提供食物热量查询、饮食记录、体重管理。

**优势**：
- 食物数据库大（百万级），热量查询准确
- 饮食记录体验好（扫码、语音、图片识别）
- 体重管理功能完善（趋势图、目标设定）

**劣势**：
- 训练功能弱，只是附属功能
- 没有 AI 对话能力
- 没有动作评估能力
- 商业模式依赖广告和会员

**对本产品的启示**：
- 薄荷健康证明了"数据记录 + 趋势分析"的用户价值
- 本产品在训练侧做了类似的事：训练记录 → 趋势分析 → 个性化建议
- 饮食管理可以作为未来扩展方向

### 4.3 Freeletics

**产品概述**：德国公司，AI 驱动的个人训练应用，主打无器械训练。

**优势**：
- AI 训练计划生成：基于用户反馈调整训练内容
- 训练适应性：根据用户的完成情况动态调整难度
- 全球化运营，多语言支持
- 训练内容质量高（专业教练设计）

**劣势**：
- AI 对话有限：不是真正的对话，而是预设选项 + 模板回复
- 无动作评估：不分析用户动作质量
- 价格高：订阅制 $34.99/3 个月
- 内容偏向无器械，力量训练覆盖不足

**对本产品的启示**：
- Freeletics 验证了"AI 驱动训练计划"的市场需求
- 但它的 AI 更像是"规则 + 模板"，不是真正的 AI 对话
- 本产品在 AI 对话和动作评估两个维度上都更有深度

### 4.4 Hevy

**产品概述**：力量训练记录工具，主打简洁的训练记录和数据可视化。

**优势**：
- 训练记录体验极好（快速添加动作、组数、重量）
- 数据可视化强（PR 记录、肌肉群分布、训练频率）
- 社交功能（关注、训练动态）
- 免费功能足够用

**劣势**：
- 纯记录工具，没有 AI 能力
- 没有训练建议，用户需要自己知道练什么
- 没有动作评估
- 无中文支持

**对本产品的启示**：
- Hevy 证明了"好的训练记录体验"本身就有价值
- 本产品的训练记录功能可以借鉴 Hevy 的简洁设计
- Hevy 的弱点（没有 AI 指导）正是本产品的核心价值

## 5. 市场空白分析

基于竞品分析，市场上存在以下未被满足的需求：

| 空白 | 描述 | 本产品如何填补 |
|------|------|--------------|
| **动作评估** | 没有产品能告诉用户"你的深蹲膝盖内扣了" | MediaPipe 姿态估计 + 多维度评分 |
| **AI 对话指导** | 现有产品都是模板回复，不是真正的对话 | DeepSeek API + RAG 知识库 |
| **伤病安全** | 没有产品在用户问伤病问题时引导就医 | 代码层预检 + 提示词红线双层防护 |
| **数据闭环** | 训练记录和 AI 建议是割裂的 | 训练→记录→分析→建议→调整的完整闭环 |
| **2D 视频分析** | 现有动作分析需要专业设备或 3D 相机 | 普通手机 2D 视频即可，通过肘角设计解决角度依赖 |

## 6. 差异化定位

```
                 AI 能力强
                    │
                    │    ★ 本产品
                    │    （AI 对话 + 动作分析 + 数据闭环）
                    │
    记录工具 ───────┼──────────── 内容平台
    （Hevy）        │              （Keep）
                    │
                    │    Freeletics
                    │    （AI 计划，无动作评估）
                    │
                 AI 能力弱
```

**一句话定位**：本产品是市场上唯一同时具备 AI 对话教练和动作视频分析能力的健身工具。

## 7. 竞争策略

| 策略 | 具体做法 |
|------|---------|
| **聚焦训练** | 不做社交、不做电商、不做内容，只做训练相关的 AI 工具 |
| **深度 AI** | 不是"AI 噱头"，而是真正有技术深度的 AI（RAG、姿态估计、规则引擎） |
| **数据闭环** | 用得越久 AI 越精准，形成用户粘性 |
| **安全第一** | 伤病安全防护是产品底线，不是功能亮点 |
| **轻量部署** | Web 应用 + Docker 容器化，不依赖 App 商店审核 |
