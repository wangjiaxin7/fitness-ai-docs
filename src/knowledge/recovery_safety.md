# 康复与安全指南

> 科学训练，预防伤病，持续进步

---

## 热身的重要性和方法

### 为什么要热身？

热身是每次训练前必不可少的环节，主要目的包括：

- **提升体温**：使肌肉温度升高 1–2°C，降低肌肉粘滞性，提高收缩效率
- **激活神经系统**：增强神经-肌肉连接，提升反应速度和协调性
- **增加关节活动度**：促进滑液分泌，润滑关节，减少摩擦损伤
- **提高血流量**：将氧气和营养物质输送到即将工作的肌肉群
- **心理准备**：帮助大脑进入训练状态，集中注意力

忽略热身直接进行高强度训练，肌肉拉伤和关节损伤的风险会显著增加。

### 热身的基本原则

| 原则 | 说明 |
|------|------|
| **渐进性** | 从低强度逐步过渡到训练强度 |
| **特异性** | 针对即将进行的训练动作设计热身 |
| **持续时间** | 一般 5–15 分钟，视环境温度和个人情况调整 |
| **出汗标准** | 达到轻微出汗即可，不宜过度消耗体能 |

### 动态拉伸（热身主力）

动态拉伸是热身阶段的核心，通过**主动、有控制地移动关节和肌肉**来达到活动目的。

**经典动态拉伸动作：**

1. **腿部摆动（Leg Swings）**——前后、左右摆动腿，激活髋关节
2. **弓箭步旋转（World's Greatest Stretch）**——弓步同时转体，活动胸椎和髋部
3. **抱膝走（Walking Knee Hugs）**——每走一步抱膝至胸前，拉伸臀大肌
4. **高抬腿（High Knees）**——激活髋屈肌和核心
5. **开合跳（Jumping Jacks）**——快速提升心率
6. **肩部环绕（Arm Circles）**——从小到大画圈，活动肩关节
7. **猫牛式（Cat-Cow）**——脊柱波浪活动，适合训练前激活核心与背部

### 静态拉伸 vs 动态拉伸

| 对比维度 | 动态拉伸 | 静态拉伸 |
|----------|----------|----------|
| **时机** | 训练前 | 训练后 |
| **方式** | 有控制的重复动作 | 保持固定位置 15–60 秒 |
| **效果** | 激活肌肉、提升活动度 | 延长肌肉、降低张力 |
| **对力量的影响** | 无负面影响，甚至有益 | 超过 60 秒的静态拉伸可能暂时降低力量输出 |
| **适用场景** | 热身 | 冷却、柔韧性训练、康复 |

> 关键提醒：**训练前不宜做长时间静态拉伸**，这会暂时降低肌肉力量输出和爆发力，增加受伤风险。静态拉伸应保留到训练后进行。

## 训练后拉伸

### 为什么要训练后拉伸？

- **促进恢复**：帮助肌肉从收缩状态恢复到静息长度
- **减轻 DOMS**：缓解延迟性肌肉酸痛的强度
- **提升柔韧性**：在有充分血流的温热状态下，拉伸效果最佳
- **降低肌肉紧张**：缓解训练带来的神经和肌肉紧绷

### 训练后拉伸建议

1. **主要肌群**：每个大肌群（胸、背、腿、肩）各选 1–2 个拉伸动作
2. **保持时间**：每个动作 20–30 秒，不超过 60 秒
3. **力度控制**：感觉到轻微牵拉感，**不要拉到刺痛**
4. **呼吸配合**：保持深呼吸，在呼气时加深拉伸幅度

### 常见拉伸动作

- **股四头肌拉伸**：站立位，脚跟拉向臀部
- **腘绳肌拉伸**：腿伸直弯腰触脚尖，或仰卧位抬腿用毛巾辅助
- **胸部拉伸**：扶门框或墙面，身体向前倾
- **下背部拉伸**：婴儿式（跪姿向前俯身）或膝盖抱胸（仰卧位）
- **肩部拉伸**：手臂横过胸前用另一只手辅助固定
- **小腿拉伸**：弓箭步推墙，后腿伸直脚跟踩实

### 泡沫轴（泡沫滚筒）辅助

训练后使用泡沫轴进行自我肌筋膜放松也是一种有效的方式：

- 每个部位缓慢滚动 30–60 秒
- 在痛点（扳机点）停留 20–30 秒，深呼吸放松
- **避免**直接滚压骨骼、关节和下背部（腰部）
- 力度适中，避免过度按压导致淤青

## 常见伤病预防

### 腰痛（下背部疼痛）

**常见原因：**
- 久坐导致髋屈肌过紧、臀肌无力
- 硬拉或深蹲时腰部代偿（圆背）
- 核心肌群薄弱，脊柱稳定性不足

**预防策略：**
- 强化核心：平板支撑、死虫式、鸟狗式
- 保持正确姿势：硬拉时保持脊柱中立，深吸气后锁定核心
- 定期站立活动：每坐 30–45 分钟起身活动 2 分钟
- 拉伸髋屈肌：弓步髋屈肌拉伸
- 激活臀肌：训练前做臀桥或弹力带侧向行走

**警示信号**：如果腰痛伴随腿部放射痛、麻木或无力，可能涉及椎间盘问题，应停止训练并就医。

### 膝痛（髌腱炎 / 跑步膝）

**常见原因：**
- 股四头肌与腘绳肌力量不平衡
- 膝盖内扣（训练时膝关节不稳）
- 跑量增加过快（10% 原则被忽略）
- 鞋履磨损或鞋型不适合足弓类型

**预防策略：**
- 强化大腿后侧：腘绳肌弯举、罗马尼亚硬拉
- 膝盖对准第二脚趾：深蹲、弓步时注意膝盖方向
- 加强臀部力量：臀肌是膝关节的"减震器"
- 合理控制跑量：每周增加不超过 10%
- 选择合适的跑鞋：根据足型和跑姿选购，每 500–800 公里更换

### 肩痛（肩袖损伤 / 肩峰撞击）

**常见原因：**
- 过头动作（卧推、实力推、引体向上）技术不当
- 肩胛骨缺乏稳定性
- 圆肩驼背的体态导致肩峰下空间狭窄
- 训练量过大，肩袖肌群疲劳

**预防策略：**
- 加强肩袖肌群：外旋训练（弹力带或侧卧哑铃外旋）
- 稳定肩胛骨：面拉、弹力带划船、Y-T-W-L 练习
- 卧推时注意：肩胛骨收紧后缩、落杠位置在胸下沿
- 避免过度训练：推拉类动作比例维持在 1:1 左右
- 纠正体态：打开胸椎、加强上背部

### 腕痛

**常见原因：**
- 俯卧撑、倒立撑等承重动作中手腕过度背屈
- 杠铃卧推时手腕过度后折（未保持中立位）
- 长时间打字或使用鼠标叠加训练负荷

**预防策略：**
- 手腕热身：握拳-张开、顺时针/逆时针绕圈各 10 次
- 使用俯卧撑架或哑铃替代，减少手腕背屈角度
- 卧推时杠铃落在手掌根部，手腕保持中立
- 加强前臂力量：腕弯举、反向腕弯举
- 适当使用护腕，但不要依赖护具替代正确的技术

## DOMS（延迟性肌肉酸痛）

### 什么是 DOMS？

延迟性肌肉酸痛通常在训练后 **24–72 小时** 出现，是肌肉微细纤维损伤和炎症反应的正常表现。新手或重新开始训练后尤为明显。

**常见症状：**
- 触摸肌肉时有酸痛感
- 肌肉僵硬，活动范围受限
- 用力时疼痛加剧
- 通常在 3–5 天内逐渐消退

### DOMS 的处理方法

**有效的方法：**

- **轻度活动**：散步、游泳、骑自行车等低强度有氧，促进血液循环
- **主动恢复训练**：以 30–50% 强度训练同一肌群，促进血流代谢
- **充足的蛋白质和水分**：帮助肌肉修复
- **睡眠**：深度睡眠是肌肉修复的黄金时间
- **泡沫轴和按摩**：适度放松筋膜
- **冷热交替**：训练后冷敷减少炎症，24 小时后热敷促进循环

**无效或争议的方法：**

- **止痛药**：布洛芬等非甾体抗炎药可能会抑制肌肉长期适应，不建议常规使用
- **高强度拉伸**：过度拉伸可能加重肌肉损伤
- **冰敷**：对 DOMS 的长期缓解效果有限，更适用于急性损伤

### 如何区分 DOMS 和拉伤？

| 特征 | DOMS | 肌肉拉伤 |
|------|------|----------|
| 出现时间 | 训练后 24–48 小时 | 训练中或训练后立即 |
| 疼痛范围 | 广泛、对称 | 局部的、明确的痛点 |
| 左右对比 | 通常双侧对称 | 常为一侧 |
| 恢复时间 | 3–5 天自愈 | 可能需要 1–6 周 |
| 活动影响 | 僵硬但能动 | 特定动作无法完成或剧痛 |

> 如果怀疑是拉伤而非 DOMS，应立即停止对该部位的训练，冰敷 15–20 分钟，并视情况就医。

## 睡眠和恢复的重要性

### 为什么睡眠是恢复的核心？

肌肉不是在训练中生长，而是在**休息和睡眠中**修复和增长。

**睡眠对训练的影响：**

- **生长激素分泌**：深度睡眠期间，生长激素分泌达到峰值，是肌肉修复的关键
- **蛋白质合成**：睡眠不足会降低肌肉蛋白质合成率
- **神经系统恢复**：睡眠帮助恢复中枢神经系统的疲劳
- **皮质醇调控**：睡眠不足导致皮质醇水平升高，促进肌肉分解
- **注意力与协调性**：睡眠不足直接影响训练时的神经控制和动作质量

### 睡眠建议

| 指标 | 推荐 |
|------|------|
| **时长** | 7–9 小时 |
| **规律性** | 每天同一时间入睡和起床 |
| **睡前准备** | 睡前 1 小时减少屏幕时间，避免高强度刺激 |
| **环境** | 黑暗、安静、凉爽（18–22°C）的卧室 |
| **饮食** | 睡前 2–3 小时避免大餐，减少咖啡因在下午后的摄入 |

### 主动恢复手段

- **低强度有氧**（散步、骑行、游泳）——促进血液循环和代谢废物排出
- **按摩和泡沫轴**——缓解肌肉紧张
- **冷热交替浴**——血管收缩与扩张交替，促进循环（• 冷 1–3 分钟 → 热 3–5 分钟，重复 2–3 轮）
- **压缩衣**——促进静脉回流，减少肌肉振动和损伤
- **冥想与深呼吸**——降低交感神经兴奋，促进副交感神经主导的恢复状态

### 营养与恢复

- **蛋白质**：每公斤体重每天摄入 1.6–2.2 克
- **碳水化合物**：训练后补充，帮助糖原恢复（尤其是耐力训练）
- **水分**：训练前、中、后持续补水，脱水 2% 就会显著影响表现
- **微量元素**：镁（放松肌肉）、锌（免疫与蛋白质合成）、维生素 D（骨骼与肌肉功能）

## 训练过度信号

### 什么是训练过度？

训练过度是指训练强度和/或容量超过了身体恢复能力，导致身体功能和运动表现持续下降的状态。与暂时的疲劳不同，训练过度需要数周甚至数月才能恢复。

### 训练过度的主要信号

#### 生理信号

- **持续疲劳**：起床后仍感觉疲惫，训练中力量下降
- **静息心率升高**：早晨静息心率比正常高出 5–10 次/分
- **睡眠质量下降**：难以入睡、易醒、醒来后仍疲劳
- **免疫力下降**：频繁感冒、喉咙痛、口腔溃疡
- **食欲减退**：明显不想吃东西
- **伤病频发**：老伤复发或不断出现新伤痛
- **体重波动**：不明原因的体重下降

#### 心理信号

- **对训练失去热情**：不想去健身房或训练
- **易怒或情绪低落**：日常情绪波动增大
- **专注力下降**：工作和训练中难以集中注意力
- **睡眠焦虑**："没睡够"的焦虑反过来影响睡眠质量

#### 训练表现信号

- **力量和耐力停滞或下降**：连续 2–3 周没有进步
- **恢复速度变慢**：组间恢复时间变长
- **技术变形**：动作控制能力下降，容易做错动作
- **心率恢复变慢**：训练后心率恢复到正常水平所需时间变长

### 应对训练过度的方法

1. **减量周（Deload Week）**：每 4–8 周安排一周减量期，容量和强度降低 40–60%
2. **完全休息**：严重过度时，休息 3–7 天可能比减量效果更好
3. **增加睡眠**：目标是每晚 9+ 小时
4. **调整营养**：增加总热量摄入，特别是碳水化合物的比例
5. **降低压力**：减少非训练压力源，训练本身是"好压力"，但无法消化额外压力
6. **区分疲劳类型**：
   - **外周疲劳**（肌肉疲劳）→ 减量、按摩、营养调整
   - **中枢疲劳**（神经系统疲劳）→ 完全休息、减少高神经负荷训练（如大重量、爆发力）

### 训练与恢复的平衡原则

```
      训练刺激
          ↓
     疲劳积累
     ↙      ↘
  适应 ←————— 恢复

训练科学的核心不是练多少，而是恢复多少。
```

### 简易自我检测

每天早上起床前测量 **静息心率** 并记录 **自觉疲劳程度**。如果连续 3 天静息心率升高 5 次/分以上，同时自觉疲劳评分偏高，应考虑主动休息或减量。

## 总结

| 主题 | 核心要点 |
|------|----------|
| **热身** | 5–15 分钟动态拉伸，针对训练内容，达到微微出汗 |
| **拉伸** | 训练前动态、训练后静态，不混淆顺序 |
| **伤病预防** | 加强薄弱环节、保持技术正确、控制训练增量 |
| **DOMS 处理** | 轻度活动、充分营养、耐心等待 3–5 天 |
| **睡眠与恢复** | 7–9 小时优质睡眠是恢复的基石 |
| **训练过度** | 关注身体信号，主动安排减量周和休息 |

> **最后提醒：** 倾听身体的声音。训练计划可以调整，但身体的健康没有替代方案。进步不是一日之功，而是科学、耐心和坚持的结果。
