# 肩部与手臂训练动作详解

## 一、肩部（三角肌）

三角肌分为前束、中束、后束，训练需要全方位覆盖。

### 1.1 肩上推举（Overhead Press）

**目标肌群**：三角肌前束、中束、肱三头肌

**动作要领（杠铃）：**
1. 站姿或坐姿，杠铃位于锁骨高度，握距略宽于肩
2. 收紧核心，臀部夹紧，保持脊柱中立
3. 呼气将杠铃推至头顶正上方，肘部不完全锁死
4. 吸气缓慢下放至锁骨位置

**常见错误：**
- ❌ 腰部过度反弓 → 核心没收紧，减轻重量
- ❌ 推起时杠铃往前飘 → 保持杠铃在肩膀正上方轨迹
- ❌ 下放过低 → 底部不要低于耳朵

**哑铃版本：** 活动范围更大，对肩关节更友好，适合初学者

### 1.2 哑铃侧平举（Dumbbell Lateral Raise）

**目标肌群**：三角肌中束（让肩膀变宽的关键动作）

**动作要领：**
1. 双手持哑铃于身体两侧，肘部微屈固定
2. 以肘部引导向两侧举起，至手臂与地面平行
3. 顶峰收缩1秒，缓慢下放
4. 重量不宜过大，注重控制

**关键细节：**
- 不要耸肩（斜方肌代偿），想象"用肘尖去碰墙"
- 举起时掌心朝下（中立握），手腕中立
- 使用轻重量（5-12kg），高质量完成15-20次

### 1.3 哑铃前平举（Dumbbell Front Raise）

**目标肌群**：三角肌前束

**动作要领：**
1. 双手持哑铃于大腿前方，肘部微屈
2. 向前举起至手臂与地面平行，掌心朝下
3. 控制下放，不要借力摆动
4. 可交替完成或同时完成

### 1.4 俯身飞鸟（Bent-Over Lateral Raise / Reverse Fly）

**目标肌群**：三角肌后束、上背部

**动作要领：**
1. 俯身至躯干与地面约45°角，膝盖微屈，背部挺直
2. 双手持哑铃自然下垂，肘部微屈
3. 向两侧举起至上臂与地面平行，挤压肩胛骨
4. 控制下放

### 1.5 面拉（Face Pull）——最好的后束动作

**目标肌群**：三角肌后束、肩袖肌群

**动作要领：**
1. 用绳索训练器或弹力带，调至脸部高度
2. 双手握住绳索两端，向后拉至面部两侧
3. 肘部向外展开，双手在耳朵两侧
4. 顶峰收缩时肩胛骨后缩

## 二、手臂（肱二头肌 & 肱三头肌）

### 2.1 杠铃弯举（Barbell Curl）

**目标肌群**：肱二头肌

**动作要领：**
1. 站姿，双手与肩同宽反握杠铃
2. 肘部固定在身体两侧，不要前后晃动
3. 呼气将杠铃弯举至肩前，顶峰挤压二头肌
4. 吸气缓慢下放至手臂接近伸直

**常见错误：**
- ❌ 身体前后摆动借力 → 减轻重量或靠墙做
- ❌ 肘部前移 → 固定肘部位置

### 2.2 哑铃交替弯举（Alternating Dumbbell Curl）

**目标肌群**：肱二头肌

**动作要领：**
1. 双手持哑铃，掌心相对
2. 弯举一侧时手腕外旋（掌心朝上），即"旋后"弯举
3. 下放同时另一侧弯举，交替进行
4. 控制节奏，不借力

### 2.3 牧师凳弯举（Preacher Curl）

**目标肌群**：肱二头肌（侧重下段）

**动作要领：**
1. 上臂放在牧师凳上，腋窝紧贴垫面
2. 反握杠铃/曲杆杠铃，弯举至肩前
3. 下放至手臂接近伸直但不完全锁死
4. 此动作借力空间小，需用较小重量

### 2.4 绳索下压（Tricep Pushdown）

**目标肌群**：肱三头肌（外侧头）

**动作要领：**
1. 双手握住绳索，肘部固定在身体两侧
2. 呼气向下压至手臂伸直，顶峰收缩
3. 吸气缓慢回放至肘部约90°
4. 身体不要前后晃动

### 2.5 窄距卧推（Close-Grip Bench Press）

**目标肌群**：肱三头肌（内侧头、全长）

**动作要领：**
1. 仰卧于平凳，握距与肩同宽或略窄
2. 下放杠铃至下胸，肘部贴近身体
3. 推起至手臂伸直，顶峰收缩三头肌
4. 注意不要过度内收肘部

### 2.6 颈后臂屈伸（Overhead Tricep Extension）

**目标肌群**：肱三头肌（长头）

**动作要领：**
1. 坐姿或站姿，双手托举哑铃于头后
2. 肘部朝上，固定不动
3. 弯曲肘部将哑铃下放至头后
4. 伸直手臂回到起始位

### 2.7 前臂训练

#### 正握腕弯举（Wrist Curl）
- 前臂放在凳上，掌心朝上，手腕伸出边缘
- 屈腕向上卷起杠铃，控制下放
- 目标：前臂屈肌群

#### 反握腕弯举（Reverse Wrist Curl）
- 掌心朝下，其余同上
- 目标：前臂伸肌群

## 三、训练建议

### 肩部训练安排

| 动作 | 组数 | 次数 | 频率 |
|------|------|------|------|
| 肩上推举 | 3-4 | 8-12 | 主项 |
| 侧平举 | 3-4 | 12-15 | 辅助 |
| 面拉/俯身飞鸟 | 3 | 12-15 | 辅助 |

### 手臂训练安排

| 动作 | 组数 | 次数 |
|------|------|------|
| 杠铃弯举 | 3-4 | 8-12 |
| 绳索下压 | 3-4 | 10-15 |
| 哑铃弯举（可选） | 3 | 10-12 |
| 窄距卧推或颈后臂屈伸 | 3 | 8-12 |

### 渐进超负荷原则
- 优先增加重量（保持动作标准）
- 其次增加次数或组数
- 最后缩短组间休息（从90秒→60秒）
