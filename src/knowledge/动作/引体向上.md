# 引体向上（Pull-up）

## 目标肌群
- **主要**：背阔肌、大圆肌、斜方肌、菱形肌
- **辅助**：肱二头肌、前臂肌群、核心肌群

## 动作步骤
1. **握杠**：正握横杆（掌心向前），握距比肩宽1.5倍。手臂完全伸直，身体自然悬挂。
2. **沉肩启动**：沉肩（肩胛骨下拉并后收）激活背部，核心收紧，双膝微屈或交叉。
3. **拉起**：呼气发力，以肘部向下向后驱动，将身体拉起至下巴过杆。想象用肘尖去够髋部。
4. **顶峰收缩**：在最高点挤压背部肌群1秒，肘部位于身体两侧或稍后方。
5. **下放**：控制离心，缓慢下放至手臂完全伸直（不突然松力），速度约2-3秒。

## 呼吸方法
- **拉起时**：呼气（可利用爆发力）
- **下放时**：吸气
- **组间休息**：充分休息2-3分钟

## 常见错误
❌ **借摆动（kipping）完成** → 新手应先练严格版
✅ **纠正**：收紧核心，双腿前置或交叉，减少摆动

❌ **耸肩代偿** → 背部发力感消失，斜方肌过度参与
✅ **纠正**：拉起前主动沉肩，保持肩胛骨下压

❌ **下放速度过快** → 离心收益丢失，训练效果打折扣
✅ **纠正**：控制下放2-3秒

❌ **半程动作（不下放到完全伸直）** → 活动范围不足
✅ **纠正**：每次下放至手臂完全伸直再拉下一次

❌ **头部过度后仰或前伸** → 颈椎压力
✅ **纠正**：保持头部中立，眼睛平视前方

## 变式动作
- **反握引体向上（Chin-up）** → 侧重肱二头肌和背阔肌下部
- **宽握引体向上** → 侧重背阔肌宽度
- **对握引体（中立握）** → 对肩关节最友好
- **负重引体向上** → 进阶加重量（挂杠铃片）
- **离心引体向上**（慢放5-8秒）→ 新人突破零的经典方法

## 注意事项
- 零基础突破：先做离心引体（跳上去慢放5秒）→ 辅助引体（弹力带）→ 标准引体
- 辅助训练：高位下拉（Lat Pulldown）是引体的最佳替代和辅助动作
- 训练频率：每周2-3次，每次3-5组
- 握力不足时使用镁粉或助力带
- 不要过度训练，背阔肌恢复时间较长（48-72小时）
