# 杠铃划船（Barbell Row）

## 目标肌群
- **主要**：背阔肌、斜方肌中下部、菱形肌
- **辅助**：竖脊肌、肱二头肌、后三角肌

## 动作步骤
1. **起始姿势**：站姿，双脚与髋同宽，正握杠铃（也可反握），握距比肩稍宽。
2. **俯身**：屈髋屈膝，俯身至躯干与地面约呈45°-70°，杠铃位于膝盖正前方，腰背挺直不弓背。
3. **拉起**：保持躯干稳定不动，呼气将杠铃沿大腿拉至下腹/肚脐位置，肘部向后驱动，肩胛骨收紧。
4. **顶峰收缩**：在最高点夹紧背部1秒，感受背阔肌和斜方肌的收缩。
5. **下放**：控制离心，缓慢下放至手臂伸直（肘不锁死），不借惯性甩动。

## 呼吸方法
- **上拉前**：深吸气憋气
- **拉起时**：呼气发力
- **离心下放时**：吸气
- **重重量时**：全程憋气完成后再换气

## 常见错误
❌ **弓背（圆背）** → 腰椎椎间盘突出风险
✅ **纠正**：挺胸收腹，保持脊柱中立，降低重量

❌ **靠身体晃动（借力）拉起** → 背部发力感弱，腰部代偿
✅ **纠正**：减轻重量，用严格的动作完成，躯干保持不动

❌ **拉起时肘部外扩** → 后束代偿过多，背阔肌刺激减弱
✅ **纠正**：肘部向身体两侧后方驱动

❌ **抬起上半身跟随** → 目标肌群受力减少，变成半程硬拉
✅ **纠正**：躯干角度全程保持不变

❌ **杠铃拉得太高（拉到胸口）** → 斜方肌代偿，背阔肌卸力
✅ **纠正**：拉到下腹/肚脐位置即可

## 变式动作
- **潘德雷划船（Pendlay Row）** → 每次触地，爆发力起，适合练力量和爆发力
- **哑铃单臂划船** → 单侧训练，行程更大，方便纠正两侧不平衡
- **T杠划船** → 对下背更友好，能上更大重量
- **反握划船** → 侧重背阔肌下部和肱二头肌
- **坐姿划船** → 器械动作，腰部支撑好，适合孤立练背

## 注意事项
- 划船是背部厚度的关键训练动作
- 新手建议从较轻重量学习动作模式，感受背部发力
- 拉起时想象用手肘去够后方，而不是用手拉
- 护具：大重量时使用助力带和腰带
- 训练频率：每周1-2次
- 配合引体向上形成推拉平衡训练（一个练宽度一个练厚度）
