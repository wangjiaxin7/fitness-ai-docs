# 常见运动损伤预防与康复指南

参考来源：ACSM《Guidelines for Exercise Testing and Prescription》、Stuart McGill《Back Mechanic》、Kelly Starrett《Becoming a Supple Leopard》

## 一、运动损伤的分类

### 按损伤性质分
| 类型 | 说明 | 典型损伤 |
|------|------|---------|
| **急性损伤** | 单次事件的剧烈损伤 | 肌肉拉伤、韧带撕裂、骨折 |
| **慢性/过度使用损伤** | 反复微损伤累积 | 腱炎、应力性骨折、滑囊炎 |

### 按损伤组织分
- **肌肉**：拉伤（Ⅰ-Ⅲ级），股后肌群最易
- **肌腱**：肌腱炎/肌腱病（如髌腱炎、跟腱炎）
- **韧带**：扭伤（如踝关节扭伤）
- **关节**：撞击综合征、关节炎
- **骨骼**：应力性骨折
- **椎间盘**：腰椎间盘突出

## 二、常见损伤与预防

### 1. 腰痛（特别是下背部）

**常见原因**：
- 久坐→髋屈肌过紧→臀肌无力（下交叉综合征）
- 深蹲/硬拉时姿势错误（圆背）
- 核心薄弱，脊柱稳定性不足
- 训练容量突然增加

**预防策略**：
- 强化核心：死虫式、鸟狗式、平板支撑（优先于卷腹）
- 髋关节活动度：弓步髋屈肌拉伸
- 每日臀肌激活：臀桥、弹力带侧向行走
- 硬拉时严格遵守脊柱中立位
- 每坐45分钟起身活动2分钟

**康复信号**：如腰痛伴随腿部放射痛、麻木或无力→可能涉及椎间盘问题→立即就医

### 2. 膝痛（髌腱炎/髌股关节疼痛综合征）

**常见原因**：
- 股四头肌与腘绳肌力量不平衡（比例应接近3:4）
- 膝盖内扣（深蹲/弓步时）
- 跑量增加过快（超过10%/周）
- 髋外展肌群薄弱

**预防策略**：
- 加强臀中肌（单腿臀桥、弹力带侧向行走）
- 深蹲时膝盖对准第二脚趾
- 避免膝关节过度前移超过脚尖
- 合理控制跑量增加（每周不超10%）
- 选择合适跑鞋（每500-800km更换）

**纠正训练**：
- 坐姿腿屈伸恢复控制（轻重量，脚尖朝外→加强股内侧肌）
- 反向北欧挺（Reverse Nordic Curl）

### 3. 肩痛（肩袖损伤/肩峰撞击）

**常见原因**：
- 过头动作（卧推、推举）技术不当
- 肩胛骨稳定性差
- 胸小肌/上斜方肌过紧，肩胛骨前倾
- 训练量过大（推拉比不均）

**预防策略**：
- 面拉 → 最佳肩部健康动作
- Y-T-W-L 训练 → 肩胛骨稳定
- 卧推时肩胛骨收紧后缩
- 推拉比维持在1:1左右（每1个推类配1个拉类）
- 矫正圆肩驼背体态

### 4. 肘痛（肱骨外/内上髁炎）

**常见原因**：
- 卧推时肘关节锁死过猛
- 引体向上握距不当
- 训练容量过大

**预防策略**：
- 锁定时肘微屈
- 前臂力量训练
- 手腕热身

### 5. 腕痛
**预防**：俯卧撑使用俯卧撑架减少背屈角度、卧推时手腕保持中立、加强前臂力量

## 三、DOMS（延迟性肌肉酸痛）

### 特点
- 训练后24-72小时出现
- 肌肉微细纤维损伤+炎症反应
- 通常3-5天消退

### 处理方法
**有效**：
- 低强度活动（散步、游泳）
- 主动恢复训练（30-50%强度）
- 充足蛋白质和水分
- 睡眠
- 泡沫轴

**无效/争议**：止痛药（布洛芬等NSAIDs可能抑制长期适应）、高强度拉伸、冰敷

### DOMS vs 拉伤鉴别
| 特征 | DOMS | 肌肉拉伤 |
|------|------|----------|
| 出现时间 | 24-48小时后 | 立即或训练中 |
| 疼痛范围 | 广泛对称 | 局部明确痛点 |
| 左右对比 | 对称 | 常为一侧 |
| 恢复时间 | 3-5天 | 1-6周 |

## 四、训练过度信号

### 生理信号
- 持续疲劳（起床仍疲惫）
- 静息心率升高5-10次/分
- 睡眠质量下降
- 免疫力下降（频繁感冒）
- 食欲减退
- 伤病频发

### 心理信号
- 对训练失去热情
- 易怒或情绪低落
- 专注力下降

### 应对方法
1. 减量周（Deload）：每4-8周安排，容量降低40-60%
2. 完全休息（严重过度时3-7天）
3. 增加睡眠（每晚9+小时）
4. 调整营养（增加总热量，特别是碳水化合物）
5. 减量压力源

## 五、RICE原则（急性损伤处理）
| 步骤 | 说明 |
|------|------|
| **Rest（休息）** | 立即停止活动 |
| **Ice（冰敷）** | 每次15-20分钟，每2-3小时一次 |
| **Compression（压迫）** | 弹性绷带包扎 |
| **Elevation（抬高）** | 抬高受伤部位高于心脏 |
