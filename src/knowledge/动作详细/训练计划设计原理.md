# 训练计划设计原理 — 完整指南

参考来源：NSCA《Essentials of Strength Training and Conditioning》、ACSM《Guidelines for Exercise Testing and Prescription》、中国体育科学学会《运动训练学》

## 一、训练计划的核心原则

### 1. FITT-VP 原则
| 要素 | 说明 | 示例 |
|------|------|------|
| **频率（Frequency）** | 每周训练次数 | 力量训练每周3-6次 |
| **强度（Intensity）** | 训练负荷大小 | 用%1RM或RPE衡量 |
| **时间（Time）** | 训练时长/组间休息 | 力量训练60-90分钟 |
| **类型（Type）** | 训练模式 | 力量/耐力/爆发力 |
| **容量（Volume）** | 总训练量 | 组数×次数×重量 |
| **进阶（Progression）** | 渐进增加负荷 | 每2周增加2.5-5% |

### 2. 渐进超负荷（Progressive Overload）
肌肉只有在受到超过当前适应水平的刺激时才会增长。渐进方式：
- **增加重量**：最直接方式，每次增加2.5-5kg
- **增加次数**：在当前重量下增加1-2次
- **增加组数**：从3组增至4-5组
- **缩短休息**：从3分钟减至90秒
- **增加频率**：从每周1次增至2次
- **提高训练密度**：相同训练量缩短时间

### 3. 周期化（Periodization）
| 类型 | 说明 | 持续时间 |
|------|------|----------|
| **大周期（Macrocycle）** | 全年训练计划 | 12个月 |
| **中周期（Mesocycle）** | 阶段性目标 | 4-12周 |
| **小周期（Microcycle）** | 每周安排 | 1周 |

### 4. 线性周期化 vs 波动周期化
- **线性周期化**：强度递增，容量递减（经典5×5→3×3→1×5方案）
- **每日波动周期化（DUP）**：同一周内交替不同强度（周一5RM、周三12RM、周五8RM）
- **区块周期化（Block Periodization）**：细分积累-转换-实现三个阶段

## 二、增肌训练计划设计

### 训练变量推荐
| 变量 | 推薦值 |
|------|--------|
| 强度 | 67-85% 1RM（6-12RM） |
| 每组次数 | 6-12次 |
| 总组数/肌群/周 | 10-20组 |
| 组间休息 | 60-90秒 |
| 动作间隔 | 48-72小时 |
| 每动作组数 | 3-5组 |
| 向心速度 | 1-2秒 |
| 离心速度 | 2-3秒 |

### 训练分化方案
**一周三分化（最推荐）**
| 日 | 训练内容 |
|----|---------|
| 周一 | 推（胸+肩前中束+三头） |
| 周三 | 拉（背+肩后束+二头） |
| 周五 | 腿（股四+臀+腘绳） |

**一周四分化**
| 日 | 训练内容 |
|----|---------|
| 周一 | 胸部+三头 |
| 周二 | 背部+二头 |
| 周四 | 肩部+腹部 |
| 周五 | 腿部 |

**一周五分化（高级训练者）**
| 日 | 训练内容 |
|----|---------|
| 周一 | 胸部 |
| 周二 | 背部 |
| 周三 | 肩部 |
| 周四 | 休息 |
| 周五 | 腿部 |
| 周六 | 手臂+腹部 |

### 动作选择顺序
1. **复合多关节动作优先**（卧推、深蹲、硬拉、引体向上）
2. **辅助动作次之**（飞鸟、划船变式、侧平举）
3. **孤立动作最后**（弯举、下压、腿弯举）

## 三、减脂训练计划设计
- **力量训练为主**：维持肌肉量，保持代谢率。使用中等重量（67-75% 1RM），高组数（4-5组），短休息（45-60秒）
- **有氧训练辅助**：每周3-5次中等强度有氧（30-45分钟），或2-3次HIIT
- **NEAT增加**：非运动活动产热（走路、爬楼梯）是减脂最被低估的因素
- **容量管理**：减脂期总训练量应保持或略低于增肌期，不可骤降

## 五、减量周（Deload）
- **频率**：每4-8周安排一次减量周
- **容量**：降低至正常容量的40-60%
- **强度**：保持或降低10-20%
- **目的**：让中枢神经系统（CNS）和关节恢复，防止过度训练
- **信号**：持续疲劳、静息心率升高、力量下降、睡眠质量下降

## 六、常见错误
- 每次训练都力竭 → CNS疲劳累积
- 忽视周期化 → 平台期提前到来
- 动作选择顺序错误 → 复合动作前孤立动作，影响神经适应
- 进阶速度过快 → 动作变形、受伤
- 忽视恢复 → 训练是刺激，恢复才是成长
