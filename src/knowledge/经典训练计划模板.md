# 经典训练计划模板

> 本文档提供经过验证的经典训练模板，适用于不同目标和经验水平。

---

## 一、选择训练计划的依据

| 经验水平 | 定义 | 建议分化 |
|---------|------|---------|
| 新手 | 系统训练<6个月 | 全身训练（Full Body）或上下肢分化 |
| 中级 | 6-18个月 | 推拉腿（PPL）或上下肢分化 |
| 高级 | >18个月 | 推拉腿、五分化、专项分化 |

**新手红利期：** 前6个月即使训练质量一般，增肌效果也较明显。但动作的准确性会影响长期发展。

---

## 二、新手计划

### 2.1 全身训练（Full Body / 每周3次）

适合时间有限、刚开始训练的人。每次训练全身主要肌群。

**周一：**

| 动作 | 组数 | 次数 | 休息 |
|------|------|------|------|
| 深蹲（或腿举） | 3 | 8-10 | 90s |
| 卧推（或俯卧撑） | 3 | 8-10 | 90s |
| 坐姿划船 | 3 | 10-12 | 60s |
| 杠铃弯举 | 2 | 10-12 | 60s |
| 绳索下压 | 2 | 10-12 | 60s |
| 平板支撑 | 3 | 30-45s | 45s |

**周三：**

| 动作 | 组数 | 次数 | 休息 |
|------|------|------|------|
| 硬拉（传统或罗马尼亚） | 3 | 6-8 | 2min |
| 肩上推举 | 3 | 8-10 | 90s |
| 引体向上（或高位下拉） | 3 | 6-10 | 90s |
| 腿举 | 3 | 10-12 | 60s |
| 侧平举 | 3 | 12-15 | 45s |
| 举腿 | 3 | 10-15 | 45s |

**周五：** 同周一，可尝试增加重量或次数

### 2.2 新手渐进方案

| 周 | 策略 |
|----|------|
| 1-2 | 掌握动作模式，轻重量，每组预留2-3次余力 |
| 3-4 | 接近力竭，每个主项尝试增加2.5-5kg |
| 5-6 | 可考虑增加1组或2-3次 |
| 7-8 | 评估进展，考虑切换中级计划 |

## 三、中级计划

### 3.1 推拉腿（PPL / Push Pull Legs / 每周6练）

最受欢迎的中级分化之一，每周6天训练，3天循环两次。

**推日（Push Day A）— 侧重胸部：**

| 动作 | 组数 | 次数 | 休息 |
|------|------|------|------|
| 杠铃卧推 | 4 | 6-10 | 2-3min |
| 上斜哑铃卧推 | 3 | 8-12 | 90s |
| 坐姿哑铃肩上推举 | 3 | 8-12 | 90s |
| 绳索飞鸟 | 3 | 12-15 | 60s |
| 侧平举 | 3 | 12-15 | 45s |
| 绳索下压 | 3 | 10-15 | 60s |

**拉日（Pull Day A）— 侧重背部宽度：**

| 动作 | 组数 | 次数 | 休息 |
|------|------|------|------|
| 引体向上（或高位下拉） | 4 | 6-10 | 2min |
| 杠铃划船 | 4 | 8-12 | 90s |
| 坐姿划船 | 3 | 10-12 | 60s |
| 面拉 | 3 | 12-15 | 45s |
| 杠铃弯举 | 3 | 10-12 | 60s |
| 前臂弯举 | 2 | 12-15 | 45s |

**腿日（Leg Day）：**

| 动作 | 组数 | 次数 | 休息 |
|------|------|------|------|
| 深蹲 | 4 | 6-10 | 2-3min |
| 罗马尼亚硬拉 | 3 | 8-12 | 2min |
| 腿举 | 3 | 10-12 | 90s |
| 腿弯举 | 3 | 10-15 | 60s |
| 提踵 | 3 | 15-20 | 45s |

**推日B / 拉日B：** 更换主项和辅助动作，搭配不同的角度/器械

## 四、经典力量计划

### 4.1 StrongLifts 5×5

适合以力量提升为主要目标的训练者，核心是5个复合动作，每组5次×5组。

| 训练A | 训练B |
|-------|-------|
| 深蹲 5×5 | 深蹲 5×5 |
| 卧推 5×5 | 肩上推举 5×5 |
| 杠铃划船 5×5 | 硬拉 1×5 |
| 引体向上 3×（做到力竭） | 引体向上 3×（做到力竭） |

**安排：** 周一A，周三B，周五A，次周周一B...以此类推

**加重方案：** 每次训练深蹲+2.5kg，其他动作+2.5kg。如果某次5×5未能完成全部次数，下次重复该重量。连续3次失败则减重10%重新开始。

### 4.2 五分化训练（Bro Split）

适合高级训练者，每天一个肌群。

| 日 | 部位 |
|---|------|
| 周一 | 胸 |
| 周二 | 背 |
| 周三 | 肩 |
| 周四 | 手臂 |
| 周五 | 腿 |
| 周六 | 休息或有氧 |
| 周日 | 休息 |

**优点：** 每个部位训练量大，适合高级训练者
**缺点：** 频率低（每个部位每周1次），对中级以下效率不如PPL

## 五、进阶技术（中级以上）

### 5.1 渐进超负荷（Progressive Overload）

增肌和力量增长的最基本原则——持续给肌肉施加超过其适应的负荷。

**实施方法（按优先级）：**

1. **增加重量：** 保持次数不变，每次加2.5-5kg
2. **增加次数：** 保持重量不变，每组多加1-2次
3. **增加组数：** 每个动作多加1组
4. **缩短休息：** 从90s → 75s → 60s
5. **提高频率：** 每周多练一次该部位

**记录建议：** 使用App或笔记本记录每个动作的重量+次数，确保逐次有数据可依

### 5.2 双进阶法（Double Progression）

先增加次数，再增加重量，循环进行。

```
阶段1：卧推 70kg × 4组 × 8次
阶段2：卧推 70kg × 4组 × 10次（达标！）
阶段3：卧推 72.5kg × 4组 × 8次（新重量）
阶段4：卧推 72.5kg × 4组 × 10次
...
```

### 5.3 周期化训练

| 周期类型 | 时长 | 目的 |
|---------|------|------|
| 微周期 | 1周 | 周训练安排 |
| 中周期 | 4-8周 | 专项适应 |
| 宏周期 | 3-6个月 | 整体目标实现 |

**线性周期化示例（8周中周期）：**

| 周 | 次数范围 | 强度 | 组数 |
|---|---------|------|------|
| 1-2 | 10-12 | 60-65% 1RM | 3 |
| 3-4 | 8-10 | 65-72% 1RM | 3-4 |
| 5-6 | 6-8 | 72-78% 1RM | 4 |
| 7 | 4-6 | 78-85% 1RM | 4-5 |
| 8 | 减量周 | 50-60% 1RM | 2 |

## 六、恢复与休息

| 要素 | 建议 |
|------|------|
| 睡眠 | 7-9小时/天，低于6小时会显著影响恢复和肌肉合成 |
| 同一肌群训练间隔 | 至少48小时 |
| 减量周（Deload） | 每4-8周安排1周，重量减至60%，组数减半 |
| 主动恢复 | 休息日做轻度活动（散步、拉伸、瑜伽） |

**训练计划切换信号：**
- 连续2周无进步（力量和维度）
- 持续疲劳、睡眠质量下降
- 训练动力明显下降
- 关节出现持续性疼痛（非肌肉酸痛）
