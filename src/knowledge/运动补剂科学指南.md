# 运动补剂科学指南

> 本指南基于科学研究证据对常见运动补剂进行评估。补剂永远不能替代基础饮食和训练。

---

## 一、补剂评估等级

| 等级 | 含义 | 建议 |
|------|------|------|
| A级 | 充分证据支持有效 | 值得购买 |
| B级 | 有限证据支持有效 | 视个人预算决定 |
| C级 | 证据不足或矛盾 | 不建议优先购买 |
| D级 | 证据显示无效或有害 | 不推荐 |

---

## 二、A级补剂

### 2.1 乳清蛋白粉

| 项目 | 说明 |
|------|------|
| 证据等级 | A（多项Meta分析支持） |
| 主要作用 | 方便补充蛋白质，促进肌肉合成 |
| 建议剂量 | 20-40g/次，训练后或两餐之间 |
| 最佳时间 | 训练后30-60分钟内（但全天蛋白质总量更重要） |
| 类型选择 | 浓缩乳清（WPC）性价比高；分离乳清（WPI）热量更低适合减脂 |

### 2.2 一水肌酸

| 项目 | 说明 |
|------|------|
| 证据等级 | A（最充分研究的运动补剂之一） |
| 主要作用 | 提升力量和爆发力输出5-10%，增加肌肉细胞水合 |
| 建议剂量 | 每天3-5g，持续服用 |
| 是否需要冲击期 | 不需要。每日3-5g持续服用3-4周可达到同样效果 |
| 是否需要循环 | 不需要，可长期服用 |
| 副作用 | 少数人出现轻微肠胃不适（可随餐服用缓解） |

---

## 三、B级补剂

### 3.1 咖啡因

| 项目 | 说明 |
|------|------|
| 证据等级 | A-B（取决于使用场景） |
| 主要作用 | 提升警觉性、延迟疲劳、可能提升力量输出 |
| 建议剂量 | 训练前30-60分钟，2-6mg/kg体重 |
| 注意事项 | 避免下午晚些使用以免影响睡眠；长期使用会形成耐受 |

### 3.2 β-丙氨酸

| 项目 | 说明 |
|------|------|
| 主要作用 | 缓冲肌肉酸中毒，提升1-3分钟高强度运动表现 |
| 建议剂量 | 每天3-6g，分次服用以减轻皮麻感 |
| 副作用 | 皮肤麻木刺痛感（无害，是正常反应） |
| 适用人群 | 高强度训练（如HIIT、大重量高次数组） |

### 3.3 柠檬酸镁

| 项目 | 说明 |
|------|------|
| 主要作用 | 改善睡眠质量、肌肉放松、支持睾酮水平 |
| 建议剂量 | 睡前200-400mg |
| 适用场景 | 睡眠质量差、训练疲劳大 |

## 四、C级补剂（非必需）

### 4.1 BCAA（支链氨基酸）

结论：如果蛋白质摄入已充足（1.6g+/kg），BCAA无额外增益。唯一适用场景是长时间空腹训练或热量缺口极大时。

### 4.2 左旋肉碱

无可靠证据显示能显著促进脂肪燃烧。口服生物利用度低。

### 4.3 谷氨酰胺

饮食中已充足摄入，额外补充无明显额外收益。

### 4.4 CLA（共轭亚油酸）

减脂效果极微（约0.05-0.1kg/周），不值得购买。

---

## 五、D级补剂（不推荐）

- 睾酮促进剂：大部分成分无效或效果极微
- 燃脂丸：含麻黄碱类成分的早已被禁，其他成分基本无效
- HMB：对未受过训练者有微弱效果，对训练有素者效果很小

## 六、实战推荐

### 核心推荐（必买）

| 优先级 | 补剂 | 月花费（估算） |
|--------|------|--------------|
| 1 | 乳清蛋白粉 | 100-200元 |
| 2 | 一水肌酸 | 30-60元 |

### 可选（看需求）

| 补剂 | 月花费 | 适用场景 |
|------|--------|---------|
| 咖啡因（黑咖啡即可） | 20-50元 | 训练前需提神 |
| 维生素D | 10-30元 | 日照不足 |
| 鱼油（Omega-3） | 40-80元 | 海鱼摄入不足 |

### 不必要购买

BCAA、谷氨酰胺、CLA、燃脂丸、睾酮促进剂、HMB

---

## 七、购买注意事项

1. 选择有第三方检测的品牌
2. 关注实际有效成分剂量而非"专利配方"
3. 国内购买推荐渠道：iHerb、天猫国际官方、品牌官网
4. 注意防伪，优先选择官方渠道
