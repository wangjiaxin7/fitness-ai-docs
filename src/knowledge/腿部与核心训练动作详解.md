# 腿部与核心训练动作详解

## 一、腿部肌群

腿部训练是全身力量的基础，也是促进睾酮分泌、提升整体代谢的关键。

### 1.1 腿举（Leg Press）

**目标肌群**：股四头肌、臀大肌、腘绳肌

**动作要领：**
1. 坐于腿举机，背部紧靠椅垫，双脚与肩同宽置于踏板上
2. 解开安全锁，缓慢下放至膝盖约90°
3. 呼气蹬起至膝盖微屈，不完全锁死
4. 全程保持背部贴紧椅垫

**不同脚位的变化：**

| 脚位 | 侧重 |
|------|------|
| 双脚与肩同宽、中位 | 整体大腿 |
| 双脚高位 | 臀大肌、腘绳肌 |
| 双脚低位 | 股四头肌下部 |
| 双脚宽距 | 内收肌、股内侧肌 |
| 双脚窄距 | 股外侧肌 |

**常见错误：**
- ❌ 下放过深（臀部抬起）→ 腰椎压力大，控制幅度
- ❌ 膝盖内扣 → 膝盖韧带受力不均，保持膝盖朝向脚尖方向
- ❌ 蹬起时锁死膝盖 → 关节压力大，保持微屈

### 1.2 腿弯举（Leg Curl）

**目标肌群**：腘绳肌（股二头肌）

**动作要领（俯卧版本）：**
1. 俯卧于腿弯举机，膝盖与器械转轴对齐
2. 脚踝后侧勾住滚轴
3. 呼气将滚轴弯向臀部，顶峰收缩
4. 控制下放至接近伸直

**坐姿版本：** 躯干稳定更好，侧重腘绳肌下段

### 1.3 腿屈伸（Leg Extension）

**目标肌群**：股四头肌

**动作要领：**
1. 坐于腿屈伸机，脚踝前侧勾住滚轴
2. 呼气伸腿至接近伸直，顶峰收缩
3. 吸气控制下放
4. 避免用爆发力猛踢

**注意：** 此动作对膝盖前十字韧带（ACL）压力较大，膝痛或有旧伤者应谨慎使用

### 1.4 保加利亚分腿蹲（Bulgarian Split Squat）

**目标肌群**：股四头肌、臀大肌、核心稳定

**动作要领：**
1. 背对平凳站立，将一只脚脚背放在凳上
2. 前脚距离平凳约两大步，躯干直立
3. 下放至前腿大腿与地面平行，后腿膝盖接近地面
4. 前脚蹬起回到起始位

**优势：** 单侧训练，纠正左右不平衡；对下背压力小

### 1.5 小腿训练

#### 站姿提踵（Standing Calf Raise）
- 站姿，肩部顶住提踵机垫
- 踮起脚尖至最高点，顶峰收缩
- 缓慢下放至脚后跟低于踏板
- 目标：腓肠肌

#### 坐姿提踵（Seated Calf Raise）
- 坐姿，大腿上放置杠铃或器械
- 踮起脚尖，顶峰收缩
- 目标：比目鱼肌（深层小腿肌）

**训练建议：** 小腿是慢肌纤维为主，需要高次数（15-25次/组）和每周2-3次频率

## 二、核心训练

核心不只是腹肌，包括腹直肌、腹内外斜肌、腹横肌、竖脊肌、骨盆底肌。

### 2.1 平板支撑（Plank）

**目标肌群**：腹横肌、核心深层稳定肌群

**动作要领：**
1. 俯卧，前臂撑地，肘部在肩正下方
2. 身体呈一条直线，从脚跟到头顶
3. 收紧核心，臀部不塌不撅
4. 保持均匀呼吸，不要憋气

**进阶：**
- 30秒 → 60秒 → 90秒
- 超过120秒收益递减，建议加难度而非延长时间
- 变式：单腿平板、侧平板、负重平板

### 2.2 卷腹（Crunch）

**目标肌群**：腹直肌上段

**动作要领：**
1. 仰卧，膝盖弯曲，双脚平踩地面
2. 双手放在耳侧（不要抱头拉脖子）
3. 呼气卷起上背，下背不离地
4. 顶峰收缩1秒，吸气缓慢回放

**对比仰卧起坐：** 卷腹对腰椎压力更小，更安全

### 2.3 举腿（Leg Raise）

**目标肌群**：腹直肌下段、髋屈肌

**动作要领：**
1. 仰卧，双手放身体两侧或垫于臀部下方
2. 双腿并拢伸直，呼气抬起至与地面垂直
3. 吸气控制下放至接近地面但不触地
4. 保持下背贴地，不要反弓

**进阶：**
- 悬垂举腿（Hanging Leg Raise）→ 更强核心刺激
- 屈膝举腿 → 初学者版本

### 2.4 俄罗斯转体（Russian Twist）

**目标肌群**：腹内外斜肌

**动作要领：**
1. 坐姿，膝盖弯曲，双脚离地
2. 上身向后倾斜约45°
3. 双手合十，左右旋转躯干
4. 可手持哑铃/药球增加难度

### 2.5 农夫行走（Farmer's Walk）

**目标肌群**：核心抗旋转稳定、握力、全身

**动作要领：**
1. 双手各持一个重哑铃或壶铃
2. 挺胸沉肩，核心收紧
3. 保持躯干直立，小步快走
4. 行走20-40米，2-3组

## 三、有氧训练

### 3.1 有氧类型对比

| 类型 | 特点 | 适合人群 |
|------|------|----------|
| LISS（低强度稳态） | 如快走、慢跑、椭圆机，心率120-140 | 减脂期、初学者、恢复日 |
| HIIT（高强度间歇） | 30秒冲刺/60秒休息循环，总时长10-20分钟 | 进阶者、时间有限的训练者 |
| MISS（中等强度稳态） | 如匀速跑步、划船机，心率140-160 | 多数训练者 |
| 稳态有氧 | 如慢跑45-60分钟 | 心肺耐力提升 |

### 3.2 心率区间指南

| 区间 | 心率（最大心率的%） | 感受 | 作用 |
|------|-------------------|------|------|
| 恢复区 | 50-60% | 很轻松，能聊天 | 恢复、基础耐力 |
| 燃脂区 | 60-70% | 稍有感觉，能说短句 | 脂肪氧化效率最高 |
| 有氧区 | 70-80% | 有挑战，说话困难 | 心肺耐力提升 |
| 无氧区 | 80-90% | 很喘，说不成句 | 乳酸阈值提升 |
| 极限区 | 90-100% | 全力冲刺 | 最大摄氧量、爆发力 |

> 最大心率 ≈ 220 - 年龄（粗略估算）
> 减脂最有效区间是60-70%（燃脂区），结合力量训练效果最佳

### 3.3 有氧安排建议

| 目标 | 每周次数 | 每次时长 | 类型 |
|------|---------|---------|------|
| 减脂 | 3-5次 | 30-45分钟 | 力量+有氧结合 |
| 心肺健康 | 3-4次 | 20-40分钟 | MISS或HIIT |
| 增肌期 | 2-3次 | 20-30分钟 | LISS（不影响恢复） |

## 四、训练建议

### 腿部训练日模板

| 动作 | 组数 | 次数 | 休息 |
|------|------|------|------|
| 深蹲（主项） | 3-4 | 6-10 | 2-3min |
| 腿举 | 3 | 8-12 | 90s |
| 腿弯举 | 3 | 10-15 | 60s |
| 保加利亚分腿蹲 | 3/侧 | 8-12 | 60s |
| 提踵 | 3 | 15-20 | 45s |

### 核心训练频率

核心肌群恢复较快，每周可训练3-5次，每次选3-4个动作：
- 平板支撑 3×45秒
- 卷腹 3×15
- 举腿 3×12
- 俄罗斯转体 3×15/侧
